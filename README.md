# UTOL Mod

UTOL の非公式 Chrome 拡張機能です。
さまざまなカスタマイズが可能です。

## 機能

### 時間割ページで適用される機能

|                                                                                |                                                                                                                                                                                            |
| ------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **「提出状況一覧」の<br>「課題・テスト・アンケート一覧」を表示**           | サイドメニューから「提出状況一覧」にアクセスしなくても、<br>時間割ページから直接課題の提出状況を確認できます。<br>提出済みの課題を非表示にすることもできます。                         |
| **時間割に予定を追加**                                                         | 時間割表に期間ごとの予定を追加できます。<br>予定は各コマに１つだけ追加でき、現段階では<br>予定の追加・削除のみ可能です。<br>この機能を有効にすると「予定を追加」ボタンが表示されます。 |
| **「お知らせ」を自動的に折りたたむ**         | 「重要なお知らせ」および「緊急のお知らせ」部分を、バナーを<br>クリックすると展開するアコーディオンメニュー形式に変更します。                                                       |
| **「集中コース等」から「東大生のための<br>ワークルール入門」を削除** |「東大生のためのワークルール入門」の項目が削除されます。                                                                                         |

### すべてのページで適用される機能

|                                                                    |                                                                                                                                                                                                                                                                                                                |
| ------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **PDF ファイルリンククリック時の<br>確認ダイアログをスキップ** | PDF ファイルへのリンクをクリックした際に表示される、<br>表示するかダウンロードするかを選択するダイアログをスキップし、<br>直接 PDFファイルを表示する（開く）ことができます。<br>ダウンロードする際は開いた後の PDFビューアでダウンロードするか、<br>リンクの右クリックメニューにアクセスしてください。 |
| **サイドメニューを自動的に隠す**                               | ページの読み込み時に自動的にサイドメニューを隠します。                                                                                                                                                                                                                                                     |
| **ヘッダー右上部の名前を非表示**                               | ヘッダーに表示される名前を非表示にし、<br>スクリーンショットへの映り込み等を防ぎます。                                                                                                                                                                                                                         |

### その他の機能

|                                      |                                                                                                                                                                                                                         |
| ------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **自動ログイン<br>（試験的）**       | ログアウトされている状態で UTOL にアクセスすると自動的にログインします。<br>セキュリティ上の問題が発生したり、UTOL が正しく動作しない可能性があります<br>（現段階では特に問題は発生しないことを確認済みです）。 |
| **不要なアイコンや<br>ボタンを削除** | 機能面で不要なアイコンやボタンを削除できます。                                                                                                                                                                          |

## 各種設定

UTOL のヘッダーに表示される UTOL Mod のボタンから設定画面を開くことができます。

各機能の有効／無効を選択できます。

## インストール方法

簡単なインストール方法は以下の通りです。

1. 上記の <a href="#dlLink">"Download (ZIP)" リンク</a> から ZIP ファイルをダウンロードします。

2. ダウンロードした ZIP ファイルを解凍します。

3. Chrome の拡張機能ページ（[`chrome://extensions/`](chrome://extensions/)）を開きます。

4. ページ右上の「デベロッパーモード」を有効にします。

5. ページ左上の「パッケージ化されていない拡張機能を読み込む」ボタンをクリックし、先ほど解凍したフォルダを選択します。

6. 拡張機能が追加されたら、UTOL にアクセスして拡張機能が有効になっていることを確認します。
